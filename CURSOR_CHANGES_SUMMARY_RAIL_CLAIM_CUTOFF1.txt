DELTA FROM PREVIOUS FS (cutoff.docx -> cutoff1.docx)
----------------------------------------------------
The earlier implementation always read LR_DT from ZTRSTLMNT_RAIL.
FS cutoff1.docx requires LR_DT retrieval based on FM input I_SRCHTYP:

  I_SRCHTYP = 2 (rail / existing claim search type gc_2):
    - LR_DT  from ZTRSTLMNT_RAIL-TKNUM
    - SHTYP, TPLST from VTTK-TKNUM

  I_SRCHTYP = 1 (road shipment search type gc_1):
    - SHTYP, TPLST from VTTK-TKNUM
    - LR_DT  from YTTSTX0002-SHNUMBER (= shipment)
    - If no YTTSTX0002 row: YTTSTX0001-SHNUMBER -> REPORT_NO,
      then YTTSTX0002-REPORT_NO -> LR_DT

  I_SRCHTYP = 3 (lc_3): cutoff block not executed (not in FS).

Steps 2-5 (ZLOG_EXEC_VAR config, SHTYP/TPLST match, date compare, Fr8First
error) are unchanged from the prior FS.


--------------------------------------------------------------------------------
REQUIREMENT 1a: I_SRCHTYP = 2 — Rail shipment LR date
--------------------------------------------------------------------------------

ADDED/MODIFIED - Branch inside LOOP AT lt_pod (cutoff block):

  IF i_srchtyp EQ gc_2.
    SELECT SINGLE lr_dt FROM ztrstlmnt_rail
      INTO lw_rail_lr_dt WHERE tknum = lw_pod-tknum.
    IF sy-subrc = 0 AND lw_rail_lr_dt IS NOT INITIAL.
      SELECT SINGLE shtyp tplst FROM vttk
        INTO CORRESPONDING FIELDS OF lw_vttk_ril
        WHERE tknum = lw_pod-tknum.
      IF sy-subrc = 0.
        lv_vttk_ok = abap_true.
      ENDIF.
    ENDIF.
  ENDIF.


--------------------------------------------------------------------------------
REQUIREMENT 1b: I_SRCHTYP = 1 — Road shipment LR date (YTTS tables)
--------------------------------------------------------------------------------

ADDED - DATA variable:
  lw_report_no TYPE yttstx0001-report_no

ADDED - Branch for road search:

  ELSEIF i_srchtyp EQ gc_1.
    SELECT SINGLE shtyp tplst FROM vttk ... WHERE tknum = lw_pod-tknum.
    IF sy-subrc = 0.
      lv_vttk_ok = abap_true.
      SELECT SINGLE lr_dt FROM yttstx0002
        INTO lw_rail_lr_dt WHERE shnumber = lw_pod-tknum.
      IF sy-subrc NE 0.
        SELECT SINGLE report_no FROM yttstx0001
          INTO lw_report_no WHERE shnumber = lw_pod-tknum.
        IF sy-subrc = 0 AND lw_report_no IS NOT INITIAL.
          SELECT SINGLE lr_dt FROM yttstx0002
            INTO lw_rail_lr_dt WHERE report_no = lw_report_no.
        ENDIF.
      ENDIF.
    ENDIF.
  ENDIF.


--------------------------------------------------------------------------------
REQUIREMENT 1c: Unified LR_DT variable and VTTK gate
--------------------------------------------------------------------------------

MODIFIED - Data type:
  lw_rail_lr_dt TYPE sy-datum   (was ztrstlmnt_rail-lr_dt; holds ZTRSTLMNT_RAIL
                                  or YTTSTX0002 LR_DT per search type)

ADDED - Data variable:
  lv_vttk_ok TYPE char1   (set when VTTK read succeeds for srchtyp 1 or 2)

ADDED - Common compare (only when lv_vttk_ok = X AND lw_rail_lr_dt filled):

  READ TABLE lt_ril_cutoff WITH KEY shtyp / transplpt
  DATE_CONVERT_TO_INTERNAL on REMARKS (fallback remarks+0(8))
  IF lw_rail_lr_dt > lv_cutoff_dt -> error RETURN
